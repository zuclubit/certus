## [1.0.0] - 2025-01-XX

### Added
- Initial release of Certus
- CONSAR file validation system
- Real-time processing with SignalR
- Azure AD authentication
- PDF, Excel, and CSV export functionality
- Interactive dashboard with Recharts
- VisionOS-inspired UI design
- Internationalization support (i18next)
- Dark/light theme support
- Comprehensive component library (Radix UI + shadcn)

### Technical Stack
- React 19 with TypeScript
- Vite 6 for build tooling
- TanStack Query for server state
- Zustand for client state
- Tailwind CSS for styling
- Playwright for E2E testing
- Vitest for unit testing
